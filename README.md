# Intro-Bio-Stat
BIOL59500IB/STAT 59800 Introduction to Biostatistics

This is a 16 week graduate/senior undergraduate level course taught at Purdue University Spring 2026. Students come in with a mixed background. Most of them do not have prior exposure to programming and statistics. An overview of the class is provided in the syllabus.
